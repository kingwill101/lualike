# `crypto` Library

The `crypto` library exposes hashing, HMAC, secure random bytes, and AES-CBC
helpers backed by Dart crypto packages.

## Overview

LuaLike registers `crypto` as a namespaced library:

```lua
local digest = crypto.sha256("hello")
```

The hashing functions return lowercase hex strings. The byte-oriented
functions return raw bytes as `Uint8List`.

## Accepted inputs

Most `crypto` entrypoints accept one of these value shapes:

- a LuaLike string
- a `LuaString`
- a `Uint8List`
- a Lua table of integers

Plain strings are encoded as UTF-8 before hashing or encryption. `LuaString`
values keep their raw byte representation.

## Hash functions

### `crypto.md5(data)`

Computes an MD5 digest and returns it as lowercase hex.

This exists mainly for compatibility with existing protocols. Do not use it
for new security-sensitive designs.

### `crypto.sha1(data)`

Computes a SHA-1 digest and returns it as lowercase hex.

This is also compatibility-oriented rather than a recommended default.

### `crypto.sha256(data)`

Computes a SHA-256 digest and returns it as lowercase hex.

### `crypto.sha512(data)`

Computes a SHA-512 digest and returns it as lowercase hex.

## HMAC

### `crypto.hmac(digest, key, message)`

Computes a keyed HMAC and returns the result as lowercase hex.

- `digest`
  A digest name such as `SHA-256`.
- `key`
  The shared secret.
- `message`
  The message to authenticate.

## Random bytes

### `crypto.randomBytes(count)`

Returns a `Uint8List` containing `count` securely generated bytes.

`count` must be between `1` and `1024`.

## AES-CBC helpers

### `crypto.aesEncrypt(key, iv, data)`

Encrypts `data` using AES-CBC with PKCS7 padding.

- `key` must be 16, 24, or 32 bytes long
- `iv` must be exactly 16 bytes long
- `data` may be a string or byte-oriented value

The return value is a `Uint8List` containing ciphertext bytes.

### `crypto.aesDecrypt(key, iv, data)`

Decrypts ciphertext previously produced by `crypto.aesEncrypt()`.

The return value is a `Uint8List` containing the plaintext bytes.

## Examples

```lua
local digest = crypto.sha256("LuaLike")
print(digest)

local key = crypto.randomBytes(16)
local iv = crypto.randomBytes(16)

local ciphertext = crypto.aesEncrypt(key, iv, "secret payload")
local plaintext = crypto.aesDecrypt(key, iv, ciphertext)

print(dart.string.bytes.fromBytes(plaintext))
print(crypto.hmac("SHA-256", "shared-key", "message"))
```
